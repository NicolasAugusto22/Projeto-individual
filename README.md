# GYMTECH
Meu primeiro projeto individual !
--

O GymTech é uma aplicação web desenvolvida com foco em auxiliar usuários na sua jornada fitness, oferecendo uma simples avaliação da composição corporal, sugestões de exercícios e criação de dietas.

# Funcionalidades:
Sistema de cadastro e login de usuários, visualização de treinos por grupo muscular, análise de desempenho e evolução via dashboard sugestões de dieta.

# Tecnologias utilizadas:
HTML5
CSS3
JavaScript
Node.js
MySQL
Git e GitHub
